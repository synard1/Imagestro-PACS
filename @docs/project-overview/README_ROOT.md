# MWL-PACS-UI

This is a frontend application for a Radiology Modality Worklist (MWL) and mini-Picture Archiving and Communication System (PACS). It is built as a single-page application using React and Vite.

## Project Structure

The project is organized into the following directories:

- `docs/`: Contains all project documentation.
  - `archive/`: Old documentation files.
  - `community/`: Community-related documents.
  - `guides/`: Guides for testing, debugging, and other processes.
  - `implementation-details/`: Detailed explanations of specific features.
  - `migrations/`: Database migration files.
  - `project-overview/`: High-level project overview documents.
  - `reports/`: Project reports.
  - `schemas/`: Database schema and related documentation.
- `scripts/`: Contains various scripts for the project.
  - `legacy-tests/`: Older test scripts.
  - `release-candidates/`: Scripts for release candidates.
  - `test-runners/`: Scripts for running tests.
- `server/`: Contains the source code for the backend server.
- `server-data/`: Contains data for the backend server.
- `src/`: Contains the frontend source code.
- `tests/`: Contains all test files.
  - `api-tests/`: API tests.
  - `crud-tests/`: CRUD tests.
  - `e2e-tests/`: End-to-end tests.
- `tests-js/`: Contains JavaScript test files.
  - `unit-tests/`: Unit tests.
- `uploads/`: Contains uploaded files.

For more detailed information, please refer to the documents in the `docs/` directory.
